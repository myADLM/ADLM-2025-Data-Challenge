Indication(s) for use: The Sentinel Plasmaproteins Cal 3x is a device intended for medical purpose for use in ceruloplasmin, kappa light chains and lambda light chains assay to establish points of reference that are used in the determination of values in the measurement of ceruloplasmin, kappa and lambda light chains in human serum and plasma.

3. Special condition for use statement(s): Prescription Use Only

. Special instrument Requirements: Automated, semi-automated, and manual clinical chemistry systems

# I. Device Description:

The Sentinel Plasmaproteins $\mathbf { C a l 3 x }$ is a liquid, ready-to-use calibrator prepared from plasmatic plasmaproteins in human-based serum. It consists of $4 \mathrm { x l ~ m L }$ bottles of aqueous material containing ceruloplasmin, kappa light chains and lambda light chains in a human serum matrix. This material is stable until the date printed on the label when stored as directed. Calibrator traceability was certified to CRM 470 (Certified Reference Material).

c. Traceability (controls, calibrators, or method):

The Sentinel Plasmaproteins Cal 3x calibrator traceability was certified to CRM 470 (Certified Reference Material). The calibrator shelf-life stability was determined by the recovery method on one lot of Plasmaproteins Cal $3 \mathbf { x }$ stored at $2 - 8 { } ^ { 0 } \mathrm { C }$ , compared with the value assigned at manufacturing time. Percent recovery was calculated for each calibrator level by dividing the result in conventional units $\mathrm { ( m g / d L ) }$ of the test calibrators by the assigned value $\mathrm { ( m g / m L ) }$ and multiplying the result by 100. Acceptance criteria is $10 0 { \pm } 1 0 \%$ . Data support a shelf life of 25 months. Claim will be 24 months.
