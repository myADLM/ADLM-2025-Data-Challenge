# L. Test Principle:

Under acidic conditions, iron $( \mathrm { F e } ^ { + + + } )$ bound to the protein transferrin is released. In the presence of the reducing agent ascorbic acid, $( \mathrm { F e } ^ { + + + } )$ is reduced to $( \mathrm { F e } ^ { + + } )$ . (Fe ++) forms a blue complex with 3-(2-pyridyl)-5,6-bis-2-(5-furyl sulfonic acid)-1,2,4- triazine, disodium salt (Ferene $^ \mathrm { \textregistered }$ ). The absorbance of the complex, measured using a bichromatic (600, 700 nm) endpoint technique, is directly proportional to the concentration of transferrin-bound iron in the serum.

# d. Detection limit:

The analytical sensitivity (limit of the blank) represents the lowest concentration of iron that can be distinguished from zero. This analytical sensitivity was defined as the mean value $( \mathrm { n } { = } 2 0 )$ ) plus two standard deviations of the Level 1 $\left( 0 \mathrm { m g / L } \right)$ IRON Calibrator. The Level $1 ( 0 \mathrm { m g / L } )$ IRON calibrator was run $\mathrm { n } { = } 2 0$ on one Dimension $^ \mathrm { \textregistered }$ RxL clinical chemistry system using one reagent lot on one instrument at both the $4 0 \mathrm { { u L } }$ and $2 5 { \mathrm { u L } }$ sample size. The analytical sensitivity of the IRON method is stated as $5 \mu \mathrm { g } / \mathrm { d L }$ using a $4 0 ~ \mu \mathrm { L }$ sample size and $5 ~ { \mu \mathrm { g } } / { \mathrm { d L } }$ using a $2 5 \mu \mathrm { L }$ sample size.

# e. Analytical specificity:

Dimension $^ \mathrm { \textregistered }$ IRON was evaluated for interference according to CLSI/NCCLS EP7-A. Bias is the difference in the results between the control sample (without the interferent) and the test sample (contains the interferent) expressed in percent. The sponsor considered bias exceeding $10 \%$ to be interference. Iron dextran at a concentration of $6 0 ~ \mathrm { u g / m L }$ increases the IRON result by $6 3 ~ \mu \mathrm { g / d L }$ at an IRON concentration of $3 6 \mu \mathrm { g / d L }$ [and by $6 9 \mu \mathrm { g / d L }$ at an IRON concentration of 131 $\mu \mathrm { g / d L }$ . The following substances do not interfere with the IRON method when present in serum in the amounts indicated. Systematic inaccuracies (bias) due to these substances are less than $10 \%$ at an iron concentration of 26 to $3 8 ~ \mu \mathrm { g / d L }$ and 118 to $1 3 6 ~ \mu \mathrm { g / d L }$ .
