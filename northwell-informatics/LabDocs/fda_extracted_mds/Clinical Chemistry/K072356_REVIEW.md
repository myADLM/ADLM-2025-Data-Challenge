# H. Intended Use:

1. Intended use(s): See Indications for use below

2. Indication(s) for use:

The S-Test Calcium Reagent is intended for the quantitative determination of calcium concentration in serum or heparin plasma using the S40 Clinical Analyzer. Calcium measurements are used in the diagnosis and treatment of parathyroid disease, a variety of bone diseases, chronic renal disease and tetany (intermittent muscular contractions or spasms).The test is intended for use in clinical laboratories or physician’s office laboratories. For in vitro diagnostic use only.

3. Special conditions for use statement(s): For Prescription Use only

4. Special instrument requirements: S40 Clinical Chemistry Analyzer

# I. Device Description:

The Calcium (CA)is a single use reagent cartridges having two reagent cells, Photometric reaction cuvette, film seal and a 2-D code label. The reagent cells contain the following reagents; Reagent 1 – Monoethanolamine buffer and Hydroxquinoline, Reagent 2 - o-Cresolphthalein complexone

# L. Test Principle:

Calcium in the sample combines with o-cresolphthalein complexone in alkaline medium to form a reddish-purple color, which is measured bichromatically at $6 0 0 / 6 6 0 \mathrm { n m }$ . The intensity of color produced is directly proportional to the calcium concentration in the sample.

c. Traceability, Stability, Expected values (controls, calibrators, or methods):

The S Test CA cartridge is factory calibrated and traceable to the NIST standard Reference material SRM915b. The 2-D barcode printed on each cartridge provides the analyzer with lot-specific calibration data.

Real time stability studies have been conducted. Protocols and acceptance criteria were described and found to be acceptable. When stored at $2 { - } 8 ~ ^ { \mathrm { { \circ } C } }$ the assay reagent is good until the expiration date.

# d. Detection limit:

The Limit of Blank and Limit of Detection were determined by running a low sample (five different samples) and a BSA sample, ( $7 . 5 \%$ in saline). Each low sample was run for 3 days, 4 replicates/day. The blank sample was run 20 replicates/day for 4 days. The testing was split between two instruments. The Limit of Detection was determined to be $2 . { \dot { 3 } } \mathrm { m g } / \mathrm { d L } \mathrm { C a } ^ { 2 + }$ .

# e. Analytical specificity:

Interference studies to determine the effects of Unconjugated Bilirubin, Hemolysis and Lipemia were performed. Seven serum pools containing approximately $1 1 . 0 \mathrm { m g / d l }$ CA were spiked with varies concentrations of unconjugated bilirubin $( 0 { - } 5 0 \mathrm { m g / d L } )$ , hemoglobin $\mathrm { ( 0 { - } 1 0 0 0 ~ m g / d L ) }$ and Intralipids $\mathrm { ( 0 { - } 2 0 0 0 ~ m g / d l ) }$ ). Sponsor states that interference is considered to be significant if the analyte result is different from the control b $\mathrm { \Delta y \pm 1 0 \% }$ .

There was no significant interference from bilirubin or Lipemia. No significant interference of hemoglobin to $5 0 0 \mathrm { m g / d L }$ . A negative bias $( 1 7 \% )$ occurred at $1 0 0 0 \mathrm { m g / d L }$ hemoglobin.
