# I. Device Description:

Dimension Vista $^ \mathrm { \textregistered }$ Protein 1 Control L (low): PROT1 CON L is a multi-analyte, low level liquid human serum based product containing human ${ \bf { \mathfrak { a } } } _ { 1 }$ -acid glycoprotein , albumin, ${ \tt a } _ { 1 }$ -antitrypsin, C3 complement, C4 complement, ceruloplasmin, haptoglobin, immunoglobulin A , immunoglobulin E, immunoglobulin G, immunoglobulin M, prealbumin, soluble transferrin receptor, transferrin, and purified S-adenosylhomocysteine.

Dimension Vista $^ \mathrm { \textregistered }$ Protein 1 Control M (medium): PROT1 CON M is a multianalyte, mid-level liquid human serum based product containing human ${ \bf { \delta } } \mathrm { \bf { q } } _ { 1 }$ -acid glycoprotein , albumin, ${ \bf { \mathfrak { a } } } _ { 1 }$ -antitrypsin, $\beta _ { 2 }$ -microglobulin, C3 complement, C4 complement, ceruloplasmin, haptoglobin, homocysteine, immunoglobulin A , immunoglobulin E, immunoglobulin G, immunoglobulin M, prealbumin, soluble transferrin receptor, transferrin, and purified S-adenosylhomocysteine.

Dimension Vista $^ \mathrm { \textregistered }$ Protein 1 Control H (high): PROT1 CON H is a multi-analyte, high level liquid human serum based product containing human ${ \bf { \mathfrak { a } } } _ { 1 }$ -acid glycoprotein , albumin, ${ \tt a } _ { 1 }$ -antitrypsin, $\beta _ { 2 }$ -microglobulin, C3 complement, C4 complement, ceruloplasmin, haptoglobin, immunoglobulin A , immunoglobulin E, immunoglobulin G, immunoglobulin M, prealbumin, soluble transferrin receptor, transferrin, and purified S-adenosylhomocysteine.

Dimension Vista $^ \mathrm { \textregistered }$ Protein 3 Control: PROT3 CON is a multi-analyte, low level lyophilized, polygeline and rabbit albumin based product containing urinary ${ \bf { \mathfrak { a } } } _ { 1 }$ - microglobulin and serum albumin of human origin.

Dimension Vista $^ \mathrm { \textregistered }$ Protein 3 Calibrator: PROT3 CAL is a multi-analyte, lyophilized, polygeline based product containing urinary ${ \tt a } _ { 1 }$ -microglobulin, cystatin C and serum albumin of human origin.

Each donor or donor unit was tested and found to be negative for human immunodeficiency virus (HIV) 1 and 2, Hepatitis B virus (HBV), and Hepatitis C virus (HCV) using either tests found to be in conformance with the In Vitro Diagnostic Directive in the EU or FDA approved tests. Because no known test can offer complete assurance of the absence of infectious agents, all human derived products should be handled with caution.

The previously cleared analytes for the calibrators and controls were cleared under k061990, k063351, k063508, k051087, k061338, k061845, k061852, k062055, k063206, k063222, k063425, k063508, k063610, k063655, k063663, k063272, and k071980.
