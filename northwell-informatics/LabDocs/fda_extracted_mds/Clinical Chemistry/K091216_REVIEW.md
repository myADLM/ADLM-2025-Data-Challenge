# I. Device Description:

The Clinitek Status $^ { \textregistered + }$ Analyzer, a device modification/enhancement to the current Clinitek Status $\textsuperscript { \textregistered }$ , is a portable analyzer. It is designed to read only Siemens Reagent Strips for Urinalysis and Clinitest $\textsuperscript { \textregistered }$ hCG tests.

Identical to the Clinitek Status Analyzer, the Clinitek Status $^ +$ analyzer is intended for the measurement of the following in urine: Albumin, Bilirubin, Blood (Occult), Creatinine, Glucose, Ketone, Leukocytes, Nitrite, pH, Protein, Protein-to-Creatinine Ratio, Albumin-to-Creatinine Ratio, Specific Gravity, Urobilinogen, and human Chorionic Gonadotropin (hCG). There are no changes in reporting ranges as a result of this design improvement.

As with the current analyzer, the Clinitek Status $^ { \textregistered + }$ is intended for use in a physician’s office or hospital laboratory. Tests performed using the Clinitek Status $^ { \textregistered + }$ analyzer are intended for in vitro diagnostic use.

The Clinitek Status $^ { \textregistered + }$ analyzer optical system, urinalysis strip analytical method, immunoassay cassette analytical method, and main test steps remain the same as Clinitek Status $\textsuperscript { \textregistered }$ . They are not impacted by this design upgrade.

Since the SPlus and SConnect (SPlus instrument with LIS connection) are instrument upgrades only, the reagents used on SPlus and SConnect are the same as the existing reagents used on the Status (predicate instrument). No changes were made to the reagents used with the Clinitek Status, nor were changes made to the instruments optics, reagent handling system, control system, or assay timing.
