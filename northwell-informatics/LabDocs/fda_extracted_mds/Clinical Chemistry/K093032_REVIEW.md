1. Intended use(s): See indication for use below.

2. Indication(s) for use:

The Triage $^ \mathrm { \textregistered }$ Total 3 Controls are assayed materials to be used with the Triage $^ \mathrm { \textregistered }$ Troponin I Test, Triage $^ \mathrm { \textregistered }$ BNP Test, Triage $^ \mathrm { \textregistered }$ Cardio2 Panel and Triage $^ \mathrm { \textregistered }$ Cardio3 Panel devices and the Triage $^ \mathrm { \textregistered }$ Meter to assist the end user in monitoring product performance.

The Triage $^ \mathrm { \textregistered }$ Total 3 Calibration Verification Set are assayed materials to be used with the Triage $^ \mathrm { \textregistered }$ Troponin I Test, Triage $^ \mathrm { \textregistered }$ BNP Test, Triage $^ \mathrm { \textregistered }$ Cardio2 Panel and Triage $^ \mathrm { \textregistered }$ Cardio3 Panel devices and the Triage $^ \mathrm { \textregistered }$ Meter to assist the end user in monitoring product performance

4. Special instrument requirements: Triage $^ \mathrm { \textregistered }$ Meter

# I. Device Description:

The Triage $^ \mathrm { \textregistered }$ Total 3 Controls consist of two levels, level 1 and level 2. The Triage $^ \mathrm { \textregistered }$ Total 3 Calibration Verification Set consists of 5 levels, Levels A, B, C, D, E. All are single-use, unit dose (approximately $0 . 2 5 { \mathrm { m l } }$ ) quality control materials prepared with concentrated purified CK-MB, troponin complex, and BNP in human EDTA plasma at defined levels. The controls are stored frozen $< - 2 0 ^ { \circ } \mathrm { C }$ . Preservatives and stabilizers are added to maintain product integrity.

All human source materials used to produce this product have been tested for HbsAg, antiHCV, HIV-1 and HIV-2 and found to be non-reactive by FDA licensed tests.

5. Expected values/Reference range:

The expected values are assigned to the Triage $^ \mathrm { \textregistered }$ Total 3 Controls and the Triage $^ \mathrm { \textregistered }$ Calibration Verification set by assaying multiple vials of Triage $^ \mathrm { \textregistered }$ Total 3 Controls (TC3) and Triage $^ \mathrm { \textregistered }$ Total 3 Calibration Verification on 20 days using multiple Triage test devices. The observed mean is used to establish the expected value range which is the mean $\pm 3$ SD. The expected values of the Triage $^ \mathrm { \textregistered }$ Total 3 Controls (TC3) and Triage $^ \mathrm { \textregistered }$ Total 3 Calibration Verification Set are lot dependent and are listed on the Expected Values (EV) card for each level

In the labeling, the sponsor recommends that each laboratory establish their own acceptable ranges.
