# H. Intended Use:

1. Intended use(s): See indications for use below.

2. Indication(s) for use:

Diamond Calibrators for Medica ISE Modules are intended for in-vitro diagnostics use to provide calibration points for the $\mathrm { N a } ^ { + }$ , $\mathrm { K } ^ { + }$ , and Cl- electrodes on the Polymeco PolyChem, and Randox Daytona instruments having the Medica ISE Module.

3. Special conditions for use statement(s): For prescription use

4. Special instrument requirements: Poly-Chem analyzer and Randox Daytona analyzer with Medica ISE Module

# I. Device Description:

Calibrant A consists of a buffered solution which contains $1 4 0 \mathrm { m m o l / L N a } ^ { + }$ , $4 \mathrm { m m o l / L } \mathrm { K } ^ { + }$ , $1 2 5 \mathrm { m m o l / L }$ Cl- and preservatives. It is packaged in a foil bag with a draw tube and covered in a corrugated box. Each foil bag contains $5 0 0 ~ \mathrm { m L }$ of solution. This device contains no human or biological materials.

Calibrant B consists of a buffered solution which contains $7 0 \mathrm { { m m o l } / L \mathrm { { N a } ^ { + } } }$ , $8 \mathrm { m m o l / L } \mathrm { K } ^ { + } .$ , 41 mmol/L Cl- and preservatives. It is packaged in a brown plastic bottle with a cap. Each plastic bottle contains $1 2 5 ~ \mathrm { m L }$ of solution. This device contains no human or biological materials.

# Value Assignment

Commercially available salts/chemical constituents are gravimetrically weighed and added to Type 1 deionized water to yield the desired concentrations. To ensure that target values are met, multiple replicates of test samples are measured at the beginning and end of the production run on multiple analyzers. Values are determined by taking the mean of multiple determinations performed on randomly selected samples from each lot. Samples are then tested against previous lots normalized to either an aqueous standard made with corresponding analyte NIST (National Institute of Standards and Technology) material or a known calibrator.

# Stability

Accelerated stability testing protocols and acceptance criteria were described and found to be adequate. Calibrant A has an estimated shelf life of 30 months when stored at $1 8 { - } 2 5 ^ { \circ } \mathrm { C }$ . Calibrant B has an estimated shelf life of 28 months when stored in the dark at $1 8 { - } 2 5 ^ { \circ } \mathrm { C }$ . On Board stability for Calibrant A is 30 days. On-Board stability for Calibrant B, after pipetting into a sample cup, is 1 hour. Real time stability testing is ongoing.
