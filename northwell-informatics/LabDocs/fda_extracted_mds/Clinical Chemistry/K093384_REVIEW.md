# H. Intended Use:

1. Intended use(s): See indications for use below.

2. Indication(s) for use:

Mission Trinity R Control is intended to be used as an assayed quality control material for monitoring the precision performance of the following analytes pH, $\mathsf { p C O } _ { 2 }$ , ${ \mathsf { p O } } _ { 2 }$ ,

$\mathrm { N a } ^ { + } , \mathrm { K } ^ { + }$ , $\mathrm { { C a } ^ { + + } }$ , $\mathrm { C l } ^ { - }$ , glucose and lactate on blood gas, electrolyte and metabolite analyzers as well as tHb, $\% 0 _ { 2 } \mathrm { H b } , \% 0 _ { 2 } \mathrm { S a t }$ , $\%$ COHb, %MetHb, $\% { \bf O } _ { 2 } { \bf C } _ { \mathrm { t } }$ & $\% \mathrm { H H b }$ on CO-oximeter instrumentation.

3. Special conditions for use statement(s): Prescription use only

Radiometer ABL Blood Gas analyzers and CO-oximeters, Radiometer OSM3 COoximeter, Ciba-Corning/Bayer Blood Gas analyzers

# I. Device Description:

Trinity R consists of a buffered solution of electrolytes, glucose, lactate, dyes and preservative. It is equilibrated with specific levels of $\mathrm { C O } _ { 2 }$ , ${ \bf O } _ { 2 }$ , and $\Nu _ { 2 }$ . It contains no human or biological materials.

Trinity R is provided in three distinct levels of $\mathsf { p H }$ , $\mathsf { p C O } _ { 2 }$ , ${ \mathsf { p O } } _ { 2 }$ , $\mathrm { N a } ^ { + }$ , $\mathrm { K } ^ { + }$ , $\mathrm { C a } ^ { + + }$ , Cl¯, glucose, lactate, tHb, $\% 0 _ { 2 }$ Hb, $\% \mathrm { H b } { \mathrm { O } } _ { 2 } { \mathrm { S a t } }$ , $\% C O H \mathrm { b }$ , $\% \mathrm { M e t H b }$ , $\% { \bf O } _ { 2 } { \bf C } \mathrm { t }$ & $\% \mathrm { H H b }$ covering the significant range of the instrument performance. It is packaged in sealed glass ampules, each containing $1 . 8 \mathrm { m l }$ of solution. Ampules are packaged in single level boxes (30 ampules of same level) or tri-level boxes (10 ampules of each level). The ampules are single use products.

Traceability

Commercially available salts/chemical constituents are gravimetrically weighed and added to deionized water to yield desired concentrations. Controls are tonometered with commercially available gas to yield the desired $\mathsf { p H }$ , $\mathsf { p C O } _ { 2 }$ , ${ \mathsf { p O } } _ { 2 }$ , levels for each control. Commercially available dyes are also added to create clinically relevant cooximeter values for each control. Lot to lot variation of the Mission Trinity R controls is determined by testing the new lot vs. the previous lot normalized to either an aqueous standard made with corresponding analyte NIST material or a known calibrator.

# Value Assignment

Multiple replicates of test samples are measured at the beginning and end of the production run on various analyzers for electrolyte, metabolite, and co-oximetry values. Multiple replicates of test samples are run on two analyzers for blood gas and $\mathsf { p H }$ values. Values are determined by taking the mean of multiple determinations performed on randomly selected samples from each lot. Ranges are assigned using pre-determined intervals. Value assignment is performed for each lot of Mission Trinity R control.

The labeling states that laboratories should establish appropriate acceptance criteria when using this product for its intended use.

# Stability

Real Time, Shelf-Life, Open Vial, and Transport Simulation testing protocols and acceptance criteria were described and found to be adequate. The Mission Trinity R controls are stable until the expiration date of two years printed on the vials when stored at $2 { - } 8 \ ^ { \circ } \mathrm { C }$ . The labeling states that for pH and blood gas values the controls should be analyzed within one minute of opening and for electrolyte measurements the controls should be analyzed within one hour of opening. This device should not be frozen.
