# L. Test Principle:

The uric acid assay is a two-part reaction. Uric acid is oxidized to allantoin by uricase with production of hydrogen peroxide (H2O2). The H2O2 reacts with 4- aminoantipyrine (4-AAP) and N-(3-sulfopropyl)-3-methoxy-5-methylanaline (HMMPS) in the presence of peroxidase (POD) to yield a quinoneimine dye. The resulting change in absorbance at $6 0 4 \mathrm { n m }$ is proportional to the uric acid concentration in the sample. The two-part (R1/R2) configuration of this assay allows reduction of interference from ascorbic acid by inclusion of ascorbic oxidase in the R1 portion of the assay.

# d. Detection limit:

The sponsor conducted a Limit of Blank (LoB), Limit of Detection (LoD) and a Limit of Quantitation (LoQ) study following the CLSI EP-17A guideline. # 24 Hour Urinary Excretion

To convert results from mg/dL to mg/day (24 hour urinary excretion)

Where:

${ \mathrm { V } } = 2 4$ hour urine volume (mL) $\mathrm { C } =$ analyte concentration (mg/dL)

24 hour excretion $=$ [(V x c) / 100] mg/day

To convert results from mmol/L to mmol/day (24 hour urinary excretion)

Where:

${ \mathrm { V } } = 2 4$ hour urine volume (mL) $\mathrm { C } =$ analyte concentration (mmol/L)

24 hour excretion $= [ \left( \mathrm { V ~ x ~ c } \right) / \mathrm { ~ } ] \mathrm { m m o l } / \mathrm { d a y }$
