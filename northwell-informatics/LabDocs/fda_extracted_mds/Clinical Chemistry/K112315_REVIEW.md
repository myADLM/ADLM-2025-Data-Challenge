The y.b.t. Pregnancy Test Strip is an immunochromatographic assay for the qualitative determination of hCG in human urine. The test is intended for use as an aid in the early detection of pregnancy.

# I. Device Description:

The y.b.t. Pregnancy Test will be marketed in two formats: cassette and test strip. The test strip kit consists of one test device and a package insert. The cassette kit consists of one test device and a disposable plastic dropper, and a package insert. The device is a qualitative assay based on immunochromatography principle used to detect levels of hCG over the cutoff value $( 2 5 \mathrm { I U } / \mathrm { L } )$ of the device. The device utilizes mouse monoclonal anti- $\boldsymbol { \cdot } \boldsymbol { \beta }$ -hCG gold conjugated antibody as a signal reagent and a goat anti- mouse anti- $\mathbf { \nabla } \cdot \mathbf { a }$ -hCG polyclonal antibody as solid phase capture antibody. The control antibodies are goat anti-mouse IgG.

# L. Test Principle:

The test utilizes monoclonal gold conjugated antibody as a signal reagent and a polyclonal antibody as solid phase capture antibody. As the urine sample flows through the absorbent sample pad, the urine reconstitutes the dried beta hCG conjugate. The hCG in the urine sample will bind to this conjugate antibody and migrate further up the membrane to the test line immobilized with capture hCG antibody. A complex is formed at this site indicated by a pink line. When the concentration of hCG is lower than $2 5 \mathrm { \Pi I U / L }$ no complex is formed and no pink line is observed.

Cross-reactivity studies were performed to evaluate the interferences from other glycoprotein hormones, hFSH, hLH, and hTSH. Human FSH $( 1 0 0 0 \mathrm { I U / L }$ , WHO $2 ^ { \mathrm { n d } }$ HMG), hLH ( $1 0 0 0 \Pi / \mathrm { L }$ , WHO 68/38), and hTSH $\mathrm { 1 0 0 0 I U / L }$ , WHO $2 ^ { \mathrm { n d } }$ HMG) were added to male urine samples at two hCG concentrations (0 and $2 5 \mathrm { I U / L }$ ). The results of these studies showed that there is no cross-reactivity at 1000 IU/L hFSH, 1000 IU/L hLH, or 1000 IU/L hTSH.

The effects of $\mathsf { p H }$ and specific gravity were evaluated by testing urine samples spiked with hCG (at 0, 25 and $1 0 0 \mathrm { I U / L } )$ at $\mathsf { p H 4 - 8 }$ or specific gravity of 1.010-1.027. No interference was observed for these $\mathsf { p H }$ or specific gravities tested.
