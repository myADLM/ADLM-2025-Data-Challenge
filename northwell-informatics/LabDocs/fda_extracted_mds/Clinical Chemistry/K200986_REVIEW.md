# B Indication(s) for Use:

The OPTI® B-Lac cassette is intended to be used for the in vitro measurement of pH, ${ \mathsf { p O } } _ { 2 }$ , $\mathsf { p C O } _ { 2 }$ , total hemoglobin (tHb), and $\%$ Saturated $\mathrm { O } _ { 2 }$ in sodium heparinized venous blood samples on the OPTI CCA-TS and OPTI CCA-TS2 platform in a clinical laboratory location.

• Measurements of blood gases $\left( \mathsf { p C O } _ { 2 } , \mathsf { p O } _ { 2 } \right)$ and blood $\mathsf { p H }$ are used in the diagnosis and treatment of life-threatening acid-base disturbances.   
• Total hemoglobin (tHb) measurement is used to determine the hemoglobin content of human blood.   
• Oxygen saturation $( \mathrm { S O } _ { 2 } )$ measurement is used to determine the oxygen capacity of the hemoglobin.

C Special Conditions for Use Statement(s): Rx - For Prescription Use Only D Special Instrument Requirements: OPTI CCA-TS/TS2 Analyzers

# IV Device/System Characteristics:

# A Device Description:

The $\mathrm { O P T I } ^ { \mathfrak { \otimes } }$ B-Lac cassette is a disposable, single use cassette that contains four sensors for in vitro quantitative measurements of ${ \mathsf { p O } } _ { 2 }$ , $\mathsf { p C O } _ { 2 }$ , pH and Lactate. The cassette includes additional laser-based measurements of total hemoglobin (tHb) and $\mathrm { S O } _ { 2 }$ . The $\mathsf { p O } _ { 2 }$ , pH, tHb, and $\mathbf { S } \mathbf { O } _ { 2 }$ sensors were not modified from the previous clearance (k093280). The B-Lac cassette is sealed in a foil pouch along with a desiccant and is marked with a barcode label that includes a lot identification number, calibration information, and expiration date.

# B Principle of Operation:

The $\mathrm { { O P T I } ^ { \mathbb { B } } }$ B-Lac cassette uses fluorescence optodes to measure the intensity of light emitted from fluorescent dyes exposed to specific analytes. The concentration of the analyte is determined by the calculation of the difference in fluorescence measured at a defined calibration point and that measured with the unknown concentration of analyte. The principles of

measurement of ${ \mathsf { p O } } _ { 2 }$ , $\mathsf { p C O } _ { 2 }$ and $\mathsf { p H }$ for the B-Lac cassette are similar to those used with existing cassette styles that are used on the OPTI CCA-TS/TS2 (k984299).

2. Linearity:

To evaluate the linearity of the B-Lac $\mathsf { p C O } _ { 2 }$ , $\mathsf { p O } _ { 2 }$ , $\mathsf { p H }$ , tHb and $\mathrm { S O } _ { 2 }$ , venous whole blood samples from different donors collected in sodium heparin tubes were tonometered with a range of $\% C O _ { 2 }$ and $\% 0 _ { 2 }$ gas mixtures to generate samples with $\mathsf { p C O } _ { 2 }$ , ${ \mathsf { p O } } _ { 2 }$ , pH, and $\mathrm { S O } _ { 2 }$ values spanning the claimed measuring ranges. For ${ \mathsf { p O } } _ { 2 }$ and $\mathsf { p C O } _ { 2 }$ the concentrations were assigned by tonometry as well as using the ABL90 FLEX analyzer. 1. Method Comparison at Altitude:

A method comparison at altitude study was conducted to evaluate the performance of the BLac cassette $\mathsf { p C O } _ { 2 }$ , ${ \mathsf { p O } } _ { 2 }$ and $\mathsf { p H }$ sensors and measured tHb and $\mathrm { S O } _ { 2 }$ parameters against the ABL90 FLEX Analyzer at a range of altitudes.
