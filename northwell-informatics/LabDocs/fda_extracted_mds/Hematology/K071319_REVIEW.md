# I. Device Description:

The control set consists of bi-level human plasma, with added purified biochemicals and preservatives. It is provided in lyophilized form for increased stability. $\mathrm { C a C l } _ { 2 }$ is the reconstituting solution.

The controls are used to verify the integrity of newly received i-STAT ACT cartridges: Kaolin ACT test (K023582) and Celite ACT test (K992571) for the instruments: i-STAT 200 Portable Clinical Analyzer (K940918) and i-STAT Model 200 Portable Clinical Analyzer (K001154).
