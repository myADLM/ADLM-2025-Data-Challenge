# L. Test Principle:

This test depends on the ability of a monoclonal antibody to bind to the surface of cells expressing discrete antigenic determinants. Specific cell staining is accomplished by incubating whole blood with the monoclonal antibody reagent. The FlowCARE TLG Reagent is a combination of four murine monoclonal antibodies, each conjugated to a specific fluorochrome and specific for different cell surface antigens.

Red blood cells are lysed with the COULTER IMMUNOPREP™ Reagent System and COULTER TQ-Prep™ Workstation or equivalent. The remaining white blood cells are analyzed by flow cytometry using a sequential gating strategy. The total WBC gate is defined in the first histogram (log SS vs. CD45-FITC) by the inclusion of all $\mathrm { C D 4 5 + }$ events with low, medium and high Side Scatter. A second histogram (CD3-PC5 vs. CD4-PE), gated on the lymphocytes defined in the first histogram, identifies $\mathrm { C D 3 + C D 4 + }$ lymphocytes by setting a region to include dual positive events. A third histogram (CD3-PC5 vs. CD8-ECD), gated on the lymphocytes defined in the first histogram, identifies $\mathrm { C D 3 + C D 8 + }$ lymphocytes by setting a region to include dual positive events. The $\mathrm { C D } 3 +$ percent is obtained from either histogram 2 or 3. The $\mathrm { C D } 3 +$ , $\mathrm { C D 3 + C D 4 + }$ and $\mathrm { D 3 + C D 8 + }$ absolute count are calculated using Flow-Count Fluorospheres. Alternatively, the absolute counts may be calculated using the $\%$ lymphocytes obtained from the fourth histogram (log SS vs. CD45-FITC) gated on the total WBC. The total WBC count from the hematology analyzer is multiplied by the $\%$ lymphocytes and the $\%$ $\mathrm { _ { 0 } C D 3 + }$ , $\mathrm { D } 3 { + } \mathrm { C D } 4 +$ or $\% \mathrm { C D 3 + C D 8 + } .$ .
