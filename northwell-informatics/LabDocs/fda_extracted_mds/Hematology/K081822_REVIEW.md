# I. Device Description:

COULTER $\textsuperscript { \textregistered }$ 6C Cell Control is a hematology quality control mixture intended to be used with automated cell counters and differential cell counters. It is prepared from stabilized human blood so that repeated measurements by an automated cell counter, or differential cell counter, can be made to monitor instrument daily performance. It consists of treated, stabilized human erythrocytes, a stabilized platelet sized component, and fixed erythrocytes that simulate leukocytes and nucleated red blood cells. Three levels of varying component concentrations (Level 1, Level 2, and Level 3) are supplied.
