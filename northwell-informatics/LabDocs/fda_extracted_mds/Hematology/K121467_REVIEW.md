# H. Intended Use:

1. Intended use(s):

Eurotrol Hct Control is an assayed hematocrit reference material, to verify the precision and accuracy of the epoc $^ \mathrm { \textregistered }$ Blood Analysis System for the measurement of hematocrit.

Eurotrol HCT Control is for in vitro diagnostic use only.

epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids is an assayed hematocrit reference material, to verify the precision and accuracy of the epoc $^ \mathrm { \textregistered }$ Blood Analysis System for the measurement of hematocrit.

epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids is for in vitro diagnostic use only.

2. Indication(s) for use: Same as intended use

3. Special conditions for use statement(s): For prescription use only

4. Special instrument requirements: epoc $^ \mathrm { \textregistered }$ Blood Analysis System, manufactured by Epocal Inc.

# I. Device Description:

Eurotrol Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids are assayed hematocrit reference materials, to verify the precision and accuracy of epoc $^ \mathrm { \textregistered }$ Blood Analysis System. Eurotrol Hct Control is an electrolyte solution with conductivity appropriate to simulate clinically relevant hematocrit concentrations. These quality control materials are prepared using pure chemicals in a physiologically buffered matrix and contain no preservatives or other additives that might adversely affect electrode measurements. Eurotrol Hct Control is available in $2 . 5 \mathrm { m L }$ ampules at three different hematocrit concentration levels (A, B, C) and is packaged in 10 ampules of the same level product per carton. epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids are available at five distinct hematocrit concentration levels $\sim 1 0$ , 20, 32, 48, and $6 5 \ \% \mathrm { { P C V } }$ . Each ampule of epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids contains $2 . 5 \mathrm { m L }$ solution and 5 ampules, i.e. 1 ampule per level including instructions for use/assay sheet.

# L. Test Principle:

Eurotrol Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids are electrolyte solutions with conductivity at three and five levels respectively, appropriate to simulate clinically relevant hematocrit concentrations useful to evaluate the measurement of the epoc $^ \mathrm { \textregistered }$ Blood Analysis System.

# M. Performance Characteristics (if/when applicable):

1. Analytical performance:

a. Precision/Reproducibility:

Data were collected internally and at two external sites across different epoc Blood Analysis systems (i.e. Epocal epoc reader and Epocal epoc cartridge). Each study site performed two runs per day in duplicate for 20 distinct days on different epoc $^ \mathrm { \textregistered }$ Blood Analysis systems using 3 lots of the 3 levels of Eurotrol Hct Control and 5 levels of epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids. Acceptance criterion was predetermined to be $\leq 9 . 0 \%$ CV for the total variance which was based on compilation of the $C V \%$ for Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids at the different levels. Results collected across the three manufactured lots of Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids demonstrated consistent recovery across multiple Epocal epoc $^ \mathrm { \textregistered }$ Blood Analysis System at multiple study sites and were within the specified assay assignment ranges.

c. Traceability, Stability, Expected values (controls, calibrators, or methods):

Value assignment:

Value assignment is performed in two steps. The initial step is a quality control calculation of gravimetric values and measurements on different instruments at Eurotrol B.V to verify if the fluid has been manufactured within acceptable ranges. Final value assignment is performed using the BGEM cards and the epoc $^ \mathrm { \textregistered }$ Blood Analysis system to obtain values for end users. The value assignment is performed for each new fluid lot of Eurotrol Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids and on existing fluid lots after release of each new epoc System sensor configuration. Assigned values are only applicable for the particular epoc System Sensor Configuration.

# Closed-vial Stability:

To demonstrate the shelf life of Eurotrol Hct Control and epoc $^ \mathrm { \textregistered }$ Hematocrit Verification Fluids, multiple ampules of batches from all levels and stored at temperatures between $1 5 { - } 3 0 ^ { \circ } \mathrm { C }$ for 3 months. Hematocrit/conductivity values are measured at the following time points (0, 3, 6, 7 months). Results are analyzed using the measurand drift analysis method according to CLSI EP25- A using a $9 5 \%$ confidence interval (CI). The acceptance criterion is based on $\pm 5 \%$ maximum allowed relative deviation from the mean. Real time stability studies are- ongoing at time of clearance; however sponsor has provided data that supports 3 months stability.

Open-ampule stability: The claimed stability for opened ampules is 15 minutes at room temperature.
