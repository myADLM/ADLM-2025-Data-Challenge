Indications For Use:

The $\mathbf { x } \mathrm { T A G } ^ { \mathfrak { B } }$ Cystic Fibrosis $3 9 \mathrm { k i t } \mathrm { v } 2$ is a device used to simultaneously detect and identify a panel of mutations and variants in the cystic fibrosis transmembrane conductance regulator (CFTR) gene in human blood specimens. The xTAG $\textsuperscript { \textregistered }$ Cystic Fibrosis 39 kit v2 is a qualitative genotyping test which provides information intended to be used for carrier testing in adults of reproductive age, as an aid in newborn screening, and in confirmatory diagnostic testing in newborns and children.

The kit is not indicated for use in fetal diagnostic or pre-implantation testing. This kit is also not indicated for stand-alone diagnostic purposes.
