# 7. Device Description

Each test kit for Eu-tTG® IgA & Eu-tTG® IgG consists of one (1) microtiter plate (12 strips with 8 microwells coated with the human recombinant tTG antigen), assay controls (positive and negative), a ready-to-use set of five (5) calibrators, Horseradish Peroxidase (HRP) goat anti-human IgA or IgG conjugate, serum diluent, Tetramethylbenzidine (TMB) enzyme substrate, stop solution, and washing solution required for the assay.

# 8. Intended Use

The $\mathrm { E u - t T G } ^ { \circledast }$ IgA is an in vitro diagnostic enzyme immunoassay for the semi-quantitative detection of IgA specific antibodies directed against tissue transglutaminase (tTG) in human serum. It is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.

The $\mathtt { E u - t T G } ^ { \mathfrak { G } }$ IgG is an in vitro diagnostic enzyme immunoassay for the semi-quantitative detection of IgG specific antibodies directed against tissue transglutaminase (tTG) in human serum. It is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.
