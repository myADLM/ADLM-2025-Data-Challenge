# 7. Device Description

Each test kit for $\mathfrak { a }$ Gliatest IgA, $\mathfrak { a }$ Gliatest IgG, $a$ GliaPep® IgA, and $\mathfrak { a }$ GliaPep® IgG consists of one (1) microtiter plate (12 strips of 8 microwells coated with purified $\mathfrak { a }$ -gliadin antigen or deamidated gliadin peptide antigen), assay controls (positive and negative), a ready-to-use set of five (5) calibrators, Horseradish Peroxidase (HRP) goat anti-human IgA or IgG conjugate, serum diluent, Tetramethylbenzidine (TMB) enzyme substrate, stop solution, and washing solution required for the assay.

# 8. Intended Use

Intended Use for $\pmb { \mathbb { Q } }$ Gliatest® IgA:

The $\mathbf { a }$ Gliatest® IgA is intended for the semi-quantitative determination of IgA antibodies directed against gliadin in human serum. The assay is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.

Intended Use for $\pmb { \mathfrak { a } }$ Gliatest® IgG:

The $\mathfrak { a }$ Gliatest® IgG is intended for the semi-quantitative determination of IgG antibodies directed against gliadin in human serum. The assay is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.

Intended Use for α-GliaPep® IgA:

The $\scriptstyle \mathbf { a } - \mathbf { G } \operatorname { l i a P e p } ^ { \circledast } .$ IgA is intended for the semi-quantitative determination of IgA antibodies directed against deamidated gliadin peptide in human serum. The assay is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.

Intended Use for $\mathbf { a } { \mathbf { - G } } { \mathbf { l i a P e p } } ^ { \mathfrak { B } } { \mathbf { I g G } }$

The $\mathrm { \mathsf { a } } { \cdot } \mathrm { G i l a P e p } ^ { \oplus } \mathrm { I g G }$ is intended for the semi-quantitative determination of IgG antibodies directed against deamidated gliadin peptide in human serum. The assay is an aid in the diagnosis of celiac disease and should be used in conjunction with other serological tests and clinical findings.
