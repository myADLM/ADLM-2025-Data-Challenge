# Device Description:

HemosIL FII & FV DNA Control is synthetic Factor II and Factor V DNA suspended in a non-infectious blood-like matrix. Each vial includes normal and mutated Factor II and Factor V DNA sequences and is configured as a heterozygous control for both Factor $\Pi ~ 2 0 2 1 0 \mathrm { G } { > } \mathrm { A }$ EY and Factor $\mathrm { ~ V ~ } 1 6 9 1 \mathrm { G } { > } \mathrm { A }$ (Leiden) mutations.

The quality control material is validated for use with Xpert™M HemosIL® FI & FV genotyping assay on the GeneXpert® Dx System and is processed exactly as whole blood samples during the extraction, amplification and detection steps.

# Device Intended Use:

HemosIL FII & FV DNA Control is intended for the quality control of the Xpert HemosIL FII & FV genotyping assay on the GeneXpert Dx System.
