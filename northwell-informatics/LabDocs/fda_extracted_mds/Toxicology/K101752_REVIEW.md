1. Intended use(s): See Indications for Use below.

2. Indication(s) for use:

The CEDIA $\textsuperscript { \textregistered }$ Multi-Drug OFT Calibrators are intended for use in the calibration of d-Amphetamine, Benzoylecgonine, Morphine, and Phencyclidine (PCP) in

human Oral Fluid when used with the CEDIA Amphetamine, Cocaine, Opiate, and Phencyclidine (PCP) OFT Assays on the MGC 240 analyzer. This in vitro diagnostic device is intended for clinical laboratory use only.

4. Special instrument requirements:

For use with CEDIA $\textsuperscript { \textregistered }$ Amphetamine, Cocaine, Opiate, and Phencyclidine (PCP) OFT Assays on the MGC240 analyzer

# I. Device Description:

The CEDIA $\textsuperscript { \textregistered }$ Multi-Drug OFT Calibrators are liquid ready-to-use solutions prepared by spiking known quantities of Amphetamine, Benzoylecgonine, Morphine, and PCP into buffer matrix. The CEDIA $\textsuperscript { \textregistered }$ Multi-Drug OFT Calibrators consists of 3 calibrator levels: Negative Calibrator, Cutoff Calibrator, and High Calibrator. The Cutoff Calibrator is used as a qualitative cutoff reference for distinguishing “positive” from “negative” samples. The volume per vial is $2 0 . 0 \mathrm { m L }$ for the Negative Calibrator and $1 0 . 0 \mathrm { m L }$ for the Cutoff and High Calibrator. Amphetamine, Benzoylecgonine, Morphine, and PCP are value assigned for use with CEDIA $\textsuperscript { \textregistered }$ OFT Assays. The Negative Calibrator is free of analyte.

# Stability

Stability testing protocols and acceptance criteria were reviewed and found to be acceptable. Stability characteristics of the CEDIA $\textsuperscript { \textregistered }$ Multi-Drug OFT Calibrators were determined using accelerated and real time closed (unopened) vial, as well as real-time opened vial studies. A closed vial shelf-life of 16 months at the recommended storage temperature $2 ^ { \circ } \mathrm { C }$ to $8 ^ { \circ } \mathrm { C }$ ) was demonstrated based on real-time studies. Open vial stability of 60 days was also demonstrated at the recommended storage temperature $2 ^ { \circ } \mathrm { C }$ to $8 ^ { \circ } \mathrm { C }$ ). The closed vial shelf-life and open vial expiration dates are listed on the box and vial labels. In the package insert, the user is directed to check the expiration dates indicated on the box label. The recommendations in the labeling are to store calibrators at $2 \mathrm { ^ { \circ } C }$ to $8 ^ { \circ } \mathrm { C }$ .
