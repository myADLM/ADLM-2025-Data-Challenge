# H. Intended Use:

1. Intended use(s): See indication(s) for use below.

2. Indication(s) for use:

The TruDiagnosis $\textsuperscript { \textregistered }$ System is an in vitro diagnostic device intended for processing and genotyping multiple genetic variants in a DNA sample utilizing on-slide PCR gel-drop microarray technology. The TruDiagnosis $\textsuperscript { \textregistered }$ System consists of the TruDx $\textsuperscript { \textregistered }$ 2000 Imager, the TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit, and the ProFlex™ PCR System using the ProFlex™ 2x Flat Sample Block.

The TruDx $\textsuperscript { \textregistered }$ 2000 Imager is an instrument intended for processing and genotyping multiple genetic variants in a DNA sample utilizing on-slide PCR gel-drop microarray technology.

The TruArray $^ \mathrm { \textregistered }$ Warfarin Sensitivity Test Kit is an in vitro diagnostic test for the detection and genotyping of the $2 \mathrm { C } 9 ^ { \ast } 2$ , $2 \mathrm { C } 9 ^ { \ast } 3$ alleles of the cytochrome P450 (CYP450) 2C9 gene locus and Vitamin K epoxide reductase C1, VKORC1, gene promoter polymorphism (-1639) from genomic DNA of human saliva samples collected using the Oragene $\textsuperscript { \textregistered }$ Dx Device (OGD-500) as an aid in the identification of patients at risk for increased warfarin sensitivity. The TruArray $^ \mathrm { \textregistered }$ Warfarin Sensitivity Test Kit is a qualitative assay for use in clinical laboratories upon prescription by the attending physician.

Hardware: Tru $\operatorname { D x @ } 2 0 0 0$ 0 is an instrument that consists of a high-intensity green light emitting diode (LED), custom optics, and a digital grayscale camera. The purpose of the TruDx $\textsuperscript { \textregistered }$ 2000 Imager is to capture a fluorescence image of the microarray after completing the test. The user inserts the TruArray $\textsuperscript { \textregistered }$ Test Slide into the TruDx $\textsuperscript { \textregistered }$ 2000 Imager and follows the on-screen prompts. The ProFlex™ PCR System with the ProFlex™ 2x Flat Sample Block, a component of the TruDiagnosis $\textsuperscript { \textregistered }$ System, is an end-point thermal cycler, specifically designed for the amplification of nucleic acids using the Polymerase Chain Reaction (PCR) process.

Assay results interpreted by the TruSpot™ software program are assigned a genotype and presented to the end user in a report format.

Test Kit: The TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit includes consumables and reagents necessary to perform multiplex on-slide PCR amplification, and fluorogenic target-specific microarray-based hybridization. The TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit consists of the TruPlex PCR Master Mix, TruPlex Warfarin Primer Mix, Formamide (deionized), TruArray Wash Buffer, and TruArray Warfarin Test Slide. The TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit is intended for use with DNA extracted from saliva collected from Oragene $\textsuperscript { \textregistered }$ Dx OGD-50 device using the Qiagen QIAamp DSP DNA Mini Kit.

# L. Test Principle:

The TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit and genomic DNA (gDNA) extracted from saliva collected using the OrageneDx OGD-500 and the Qiagen QIAamp DSP DNA Mini Kit are used to perform on-slide PCR and hybridization within the TruArray $\textsuperscript { \textregistered }$ Test Slide using the ProFlex™ PCR System using the ProFlex™ 2x Flat Sample Block followed by imaging on the TruDx $\textsuperscript { \textregistered }$ 2000 Imager.

Amplification reactions are prepared using the TruArray $\textsuperscript { \textregistered }$ Warfarin Sensitivity Test Kit components (TruPlex Master Mix, TruPlex Warfarin Primer Mix and Formamide (deionized)) and extracted gDNA. Amplification reactions are loaded into the inlet port on the TruArray $^ \mathrm { \textregistered }$ Test Slide.

PCR amplification and microarray hybridization occur simultaneously in the same reaction chamber in the TruArray $\textsuperscript { \textregistered }$ Warfarin Test Slide on the ProFlex™ PCR System using the ProFlex™ 2x Flat Sample Block provided by Akonni with the TruDiagnosis $\textsuperscript { \textregistered }$ System. DNA is amplified and hybridized using a specified thermal cycling protocol. After the amplification and hybridization process, the array is washed by piercing the inlet port with a pipettor tip and dispensing $1 0 0 0 \mu \mathrm { L }$ of the TruArray $\textsuperscript { \textregistered }$ Wash Buffer through the chamber. The excess PCR amplicons, reaction by-products, and the wash buffer are retained on-slide in the integrated waste chamber. The TruArray $\textsuperscript { \textregistered }$ Test Slide is placed in a $5 0 \mathrm { m L }$ conical tube and centrifuged at ${ 5 4 0 } \mathrm { x } \mathrm { g }$ for 2 minutes in a non-refrigerated centrifuge to dry the array chamber prior to imaging.

The TruArray $\textsuperscript { \textregistered }$ Tests Slides are prepared using TruArray $\textsuperscript { \textregistered }$ microarray technology. The TruArray $\textsuperscript { \textregistered }$ is a spatially ordered array of three-dimensional hydrogel droplets, each droplet containing a unique capture probe. The hydrogel matrix is covalently attached to a glass substrate and oligonucleotide probes are covalently attached to the hydrogel scaffold. The purpose of the TruArray $^ \mathrm { \textregistered }$ is to specifically detect fluorescently labeled target amplicons via microarray-based hybridization.

Results from the TruDiagnosis $\textsuperscript { \textregistered }$ System were compared to the genotype for each sample as determined by bidirectional Sanger sequencing.

# Exogenous Interferents

Potentially interfering exogenous substances (eating, drinking, chewing gum, mouthwash and smoking) introduced into saliva samples collected using the Oragene $\textsuperscript { \textregistered }$ Dx collection device OGD-500 through various activities were tested. Each activity group was composed of five donors who each provided two samples: a baseline/control sample prior to the activity, and a sample collected 30 minutes after the activity. DNA was extracted using the Qiagen QIAamp DSP DNA Mini Kit. Samples were tested in triplicate. Results from the TruDiagnosis $\textsuperscript { \textregistered }$ System were compared to the genotype for each sample as determined by bidirectional Sanger sequencing.

Results from the testing are summarized below. There was $100 \%$ agreement between the TruDiagnosis $\textsuperscript { \textregistered }$ System and bidirectional DNA sequencing for all activities following a first run, or a second run for no calls observed during the first run, supporting that these activities have no effect on genotyping after 30 minutes.
